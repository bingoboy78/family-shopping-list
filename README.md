# 🛒 Family Shopping List | רשימת קניות משפחתית | Семейный список покупок

A modern, real-time collaborative shopping list application designed for families. Built with React, Vite, and Supabase.

---

## 🇺🇸 English

### Features
- **Real-time Sync**: Items added or toggled by one family member appear instantly on all other devices.
- **Modern UI**: Beautiful Glassmorphic design with smooth animations.
- **Collaborative**: Multiple family members can use the same list simultaneously.

### Tech Stack
- **Frontend**: React + Vite
- **Styling**: Vanilla CSS (Modern CSS Variables & Glassmorphism)
- **Backend**: Supabase (PostgreSQL + Realtime)

### Setup
1. Clone the repository.
2. Run `npm install`.
3. Create a `.env` file with your Supabase credentials (see `.env.example`).
4. Run `npm run dev`.
