# Family Shopping List / Семейный список покупок / רשימת קניות משפחתית

A mobile-first, real-time shared shopping list for families. Built with React, Vite, and Supabase.

---

## 🇺🇸 English

### Features
- **Real-time Sync:** See items added by other family members instantly.
- **Family Codes:** Join or create a list using a simple code (e.g., "SMITHS").
- **Voice Input:** Add items using your voice (supports English, Russian, and Hebrew).
- **Categorization:** Items are automatically grouped by category (Dairy, Bakery, Fruits, etc.).
- **Mobile First:** Designed to work perfectly on smartphones as a PWA.

### Setup
1. Clone the repository.
2. Install dependencies: `npm install`
3. Set up environment variables in `.env` (Supabase URL and Key).
4. Run locally: `npm run dev`

## 🇷🇺 Русский

### Особенности
- **Синхронизация в реальном времени:** Мгновенно видите товары, добавленные другими членами семьи.
- **Семейные коды:** Присоединяйтесь или создавайте список, используя простой код (например, "ИВАНОВЫ").
- **Голосовой ввод:** Добавляйте товары голосом (поддерживаются английский, русский и иврит).
- **Категории:** Товары автоматически группируются по категориям (Молочные продукты, Выпечка, Овощи и т.д.).
- **Оптимизировано для мобильных:** Работает как PWA, идеально подходит для смартфонов.

### Установка
1. Клонируйте репозиторий.
2. Установите зависимости: `npm install`
3. Настройте переменные окружения в `.env` (Supabase URL и Key).
4. Запустите локально: `npm run dev`
